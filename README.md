# Jekyll Starter

Esta es una base para iniciar tu proyecto con Jekyll, hace uso de:

- Gulp
- BrowserSync
- Sass

Requisitos:

- Ruby 2.5.0 o mayor
- NodeJS
- Jekyll

## Instalando dependencias

Para las dependencias de Jekyl:

```shell
gem install jekyll bundler
```

Para las dependencias de npm:

```shell
npm install
```

Para desarrollar:

```shell
npm start

```
